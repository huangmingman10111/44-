# 44隻石獅子
## 馬祖的石獅子
* 北竿芹壁天后宮的公石獅
  * 依據2019年連江縣政府委託案《連江縣縣定古蹟「芹壁天后宮」修復及再利用調查研究成果報告書》，北竿芹壁天后宮的立面裝飾多與北竿當地漁民生活環境息息相關的元素。 獅子兇悍威猛，廟宇建築多安座石獅作為驅凶避邪之用。芹壁天后宮正殿立面的石獅笑容滿面，具有吉祥以及歡迎訪客的意思。公獅子有典型閩南風格，採圓雕技法雕刻而成，前腳掬繡球及飄帶，象徵祈求功名與長壽。下方之石雕分別為「官印」代表封官進爵、「芭蕉」代表綿長、「棋盤」則為君子四藝之一。
  * [https://matsumemory.tw/articles/18142]
* 芹壁天后宮的母石獅
  * 依據2019年連江縣政府委託案《連江縣縣定古蹟「芹壁天后宮」修復及再利用調查研究成果報告書》，北竿芹壁天后宮的立面裝飾多與北竿當地漁民生活環境息息相關的元素。 獅子兇悍威猛，廟宇建築多安座石獅作為驅凶避邪之用，也具有祥和意象。芹壁天后宮正殿立面的母獅前腳撫逗小獅，顯示出母愛的精神，與公獅一同扭頭內望正殿的中軸線。母石獅下方之石雕分別為「畫軸」、「毛筆」與「書本」，皆代表中國傳統文人的才藝，亦寓有文人淡泊處世之精神。
  * [https://matsumemory.tw/articles/18143]
* 大王獅─南竿金板境天后宮
  * 本物件為南竿金板境天后宮內的重要文物─大王獅（公），一公一母為一對。依據廟方副主委口述，早期的大王獅沒有步伐，是由前人自創後傳給後人。後馬祖引進廣東獅，由基隆師傅前來授課。 外型兇猛，橫眉豎目，懷有獠牙，下巴有著七彩的鬍鬚，並於額頭上刻有「王」字，據推斷約為20年歷史（待確認）。現保存於金板境天后宮之東廂房內，作為天后宮之歷史展示。 2019年8月透過連江縣文化工讀進行金板境天后宮內的簡易文物調查，進行該文物的尺寸年代、文物之文字記錄、出處來源與保存現況等面向清查，可得知以下紀錄： 1.尺寸：整體─長375公分、寬69公分、高35公分；獅頭─直徑52.5公分、高34公分。 2.文物文字記錄：獅頭上有太極圖騰、王。 3.文物落款紀錄不詳。 4.保存現況穩定，獅頭局部掉漆，２只耳朵掉毛，獅身的布面局部破損。
  * [https://matsumemory.tw/articles/18235]

* 芹壁天后宮綠色石獅
  * 芹壁天后宮下方廣場一對有數十年歷史的石獅模樣很討喜，綠色的獅身在馬祖是首見，因為與戰地連結，草綠色的石獅又被稱為「軍獅」。芹壁地中海民宿老板陳天歲說，這對水泥獅是在1968年時軍方贈建的，芹壁70雄獅坑道據點開鑿完成，當時的師長有感於天后宮保佑讓工程順利完工，希望贈送一對獅子給廟裡；早年物力維艱，軍方沒能力刻石獅，只能用水泥來雕塑，但兵工從沒做過水泥獅子，不知從何下手，於是陳天歲提供了一張舞龍舞獅的照片給軍方參考，後來阿兵哥就看著照片做出了這對水泥獅，所以這對喜感十足的石獅看起來跟現在民俗技藝團的舞獅頭很像。其實這對石獅最原始的顏色並不是現在看到的綠色而是水泥原色，後來廟方信眾漆成黃色，但芹壁天后宮供奉的鐵甲元帥是青蛙神，青蛙是綠色的，所以又改成現在的綠色。這對獅子現在成為導遊介紹芹壁風情的重點，配合龜島背上的石獅石景，讓青蛙、烏龜和獅子在芹壁成為一家人。
  * [https://matsumemory.tw/articles/18444]
  * [https://www.facebook.com/memoryofmatsu/posts/340008186911534?ref=embed_post]
